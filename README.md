# Comparador Inteligente de Planilhas

Software desktop moderno para comparação inteligente de planilhas Excel.

## Funcionalidades

- Comparação automática de planilhas
- Dashboard interativo
- Gráficos modernos
- Exportação de relatórios
- Tema dark/light
- Build automático via GitHub Actions
- EXE portable

## Tecnologias

- Python
- PySide6
- Pandas
- Plotly
- OpenPyXL
- PyInstaller

## Executando

```bash
pip install -r requirements.txt
python app/main.py
